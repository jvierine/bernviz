# bernviz

Interactive Sanya ISR magnetic-meridian visualization for the Bernstein-mode figure work.

## Run locally

```bash
conda run -n base python -m http.server 8017
```

Open:

```text
http://127.0.0.1:8017/interactive_sanya_isr.html
```

## Regenerate IGRF asset

```bash
conda run -n base python export_sanya_igrf_fieldlines.py
```

## Provenance

- Earth texture: NASA Blue Marble Next Generation.
- Magnetic field lines and Sanya perpendicular TX beam: IGRF14 via `ppigrf`, generated by `export_sanya_igrf_fieldlines.py`.
- CAS logo: `https://juha.no/sanya_head_echo/scripts/caslogo.png`.
